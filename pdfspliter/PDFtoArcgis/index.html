Extrator de Coordenadas

Extração de Shapefiles (Polígono + Vértices) para ArcGIS e QGIS.

Instruções

1. Clique em "Escolher Matrícula (PDF)" para carregar um memorial descritivo em PDF.
2. Aguarde a leitura do documento e selecione a matrícula desejada em "Documentos detectados".
3. Verifique o sistema de coordenadas detectado e, se necessário, use o modo avançado para forçar o CRS.
4. Revise a validação topológica e a tabela de vértices antes de exportar.
5. Use "CSV" para baixar a matrícula selecionada ou "Salvar na Pasta" para exportar SHP + CSV.
6. Ferramenta inversa (memorial descritivo): use "Carregar SHP (ZIP)" com o shapefile completo em .zip (mínimo: .shp, .shx, .dbf e .prj).
